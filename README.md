**Arrays-strings** (Total = 10)
Brute+Optimal
  1- Find Closest number To Zero
  2- Is subsequence
  3- Summary Ranges
  4- Longest Common Prefix
  5- Best Time to Buy and Sell Stock

**Binary-Trees** (Total = 14)
Brute+Optimal
  1- Invert Binary Tree
  2- Kth Smallest Element in a BST
  3- Maximum Depth of Binary Tree
  4- Diameter of Binary tree
  5- Validate Binary Tree
  6- Implement Trie
  7- Minimum Absolute Difference

  **Hash-tables** (Total = 10)
Brute+Optimal
  1- Contains Duplicates
  2- Group Anagrams
  3- Maximum Number of Balloons
  4- Two Sum
  5- Valid Anagram

**Heaps** (Total = 6)
Brute+Optimal:
  1- Last Stone Weight
  2- Kth Largest Element in Array
  3- Top K Frequent Elements

**Linked-Lists** (Total = 6)
Brute+Optimal
  1- Linked list Cycle
  2- Merge 2 sorted Lists
  3- Remove Duplicates from Sorted list

**Two-pointers** (Total = 6)
